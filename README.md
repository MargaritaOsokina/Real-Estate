Прогнозирование финансового дистресса компаний сектора недвижимости
Описание проекта

Цель данного проекта — оценить способность традиционных финансовых коэффициентов прогнозировать финансовый дистресс публичных компаний сектора недвижимости.

На основе финансовой отчетности 270 компаний был сформирован панельный датасет за период 2014–2025 гг.. На его основе были рассчитаны финансовые коэффициенты и построены модели прогнозирования вероятности финансового дистресса компаний в следующем периоде.

В работе используются методы эконометрического анализа и машинного обучения для оценки прогностической силы финансовых показателей.

Данные

Датасет содержит финансовую информацию по 270 публичным компаниям сектора недвижимости.

Используются следующие типы данных:

показатели баланса

показатели отчета о прибылях и убытках

показатели денежных потоков

рыночные показатели

Каждое наблюдение представляет собой компания–год.

Пример структуры данных:

Компания	Год	Финансовые показатели	Финансовые коэффициенты	Distress

Целевая переменная показывает, возникает ли финансовый дистресс в следующем году.

Определение финансового дистресса

Финансовый дистресс определяется на основе финансовых показателей, отражающих способность компании:

обслуживать долговые обязательства

генерировать операционные денежные потоки

сохранять финансовую устойчивость

Модель прогнозирует вероятность события:

Distress(t+1)

на основе финансовых данных года t.

Финансовые коэффициенты

В исследовании используются следующие финансовые коэффициенты.

Долговая нагрузка

Debt to Assets

Debt_to_Assets = Total Debt / Total Assets
Ликвидность

Current Ratio

Current_Ratio = Current Assets / Current Liabilities
Прибыльность

Return on Assets

ROA = EBIT / Total Assets
Денежные потоки

Operating Cash Flow to Debt

OCF_to_Debt = Operating Cash Flow / Total Debt
Способность обслуживать долг

Interest Coverage

Interest_Coverage = EBIT / Interest Expense
Рост бизнеса

Revenue Growth

Revenue_Growth = (Revenue_t − Revenue_{t-1}) / Revenue_{t-1}
Предобработка данных

На этапе подготовки данных были выполнены следующие шаги:

преобразование финансовых показателей в числовой формат

удаление наблюдений с пропущенными значениями коэффициентов

ограничение выбросов

расчет финансовых коэффициентов

формирование индикатора финансового дистресса

создание целевой переменной Distress_next_year

Итоговый датасет для моделирования содержит примерно 2700 наблюдений.

Разведочный анализ данных (EDA)

На этапе разведочного анализа были выполнены:

анализ распределений финансовых коэффициентов

сравнение distressed и non-distressed компаний

анализ корреляций

визуализация ключевых финансовых показателей

Анализ показывает, что компании, находящиеся в финансовом дистрессе, как правило характеризуются:

более низкой прибыльностью

более слабым денежным потоком

более низкой способностью обслуживать процентные платежи

Используемые модели
Логистическая регрессия

Логистическая регрессия используется в качестве базовой эконометрической модели.

Она оценивает вероятность финансового дистресса на основе финансовых коэффициентов.

Random Forest

В качестве модели машинного обучения используется алгоритм Random Forest.

Он позволяет учитывать нелинейные зависимости между финансовыми показателями и вероятностью дистресса.

Оценка качества моделей

Качество моделей оценивается с использованием следующих метрик:

F1-score

ROC-кривая

AUC

матрица ошибок

ошибки I и II рода

Основной метрикой качества является F1-score.

Результаты моделей

Наилучшие результаты показала модель Random Forest.

Метрика	Значение
AUC	~0.94
F1-score	~0.69

Модель успешно выявляет большинство компаний, находящихся в финансовом дистрессе, при относительно низком уровне ложных срабатываний.

Анализ ошибок модели

Матрица ошибок позволяет оценить ошибки классификации.

Ошибка I рода

Модель ошибочно классифицирует финансово устойчивую компанию как distressed.

Ошибка II рода

Модель не распознаёт компанию, находящуюся в состоянии финансового дистресса.

Для задач прогнозирования дистресса более критичными считаются ошибки II рода, поскольку они означают пропуск проблемных компаний.

Интерпретация модели

Для интерпретации модели были использованы следующие методы:

анализ важности признаков

Partial Dependence Plots

SHAP-анализ

Эти методы позволяют определить, какие финансовые показатели оказывают наибольшее влияние на вероятность финансового дистресса.

Основные факторы финансового дистресса

Наиболее значимыми предикторами оказались:

Interest Coverage

ROA

OCF_to_Debt

Current Ratio

Revenue Growth

Интересно, что показатель долговой нагрузки (Debt_to_Assets) оказывает значительно меньшее влияние по сравнению с показателями прибыльности и денежных потоков.

Это означает, что вероятность финансового дистресса определяется прежде всего способностью компании генерировать прибыль и обслуживать долг, а не только уровнем долговой нагрузки.

Итоговый панельный датасет

В рамках проекта был сформирован итоговый датасет, содержащий:

исходные финансовые показатели

рассчитанные финансовые коэффициенты

вероятность финансового дистресса

бинарный прогноз модели

год прогноза

Каждая строка датасета представляет прогноз:

финансовый дистресс в году t+1
на основе финансовых данных года t
Рейтинг риска компаний

На основе предсказанных вероятностей дистресса был сформирован рейтинг компаний по уровню финансового риска.

Это позволяет выявить компании с наибольшей вероятностью возникновения финансовых проблем в будущем.

Выводы

Результаты исследования показывают, что традиционные финансовые коэффициенты обладают высокой прогностической силой при прогнозировании финансового дистресса компаний сектора недвижимости.

Наиболее важными факторами являются показатели:

способности обслуживать долговые обязательства

прибыльности

операционных денежных потоков

Модели машинного обучения, такие как Random Forest, позволяют значительно повысить точность прогнозирования по сравнению с базовыми эконометрическими моделями.
